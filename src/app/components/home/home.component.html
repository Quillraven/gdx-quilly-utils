This site contains several utilities that I regularly use for my LibGDX 2D games.

Feel free to use them as well! :)

Following utilities can be selected from the menu:

  • Tile Extruder: extrudes tiles in tilesets to avoid texture bleeding issues.
  • Spritesheet Splitter: splits a spritesheet into several tiles of equal size.
  • Image Combiner: combines multiple images into a grid layout.
  • Gradle Kotlin Template: a modern Gradle Kotlin DSL based template for LibGDX Kotlin projects.
